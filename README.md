# 📜 SwiftUI ScrollViews
A SwiftUI reference project covering ScrollView — vertical, horizontal, paging, and scroll position tracking.

---

## 🤔 What this is
This project demonstrates how to use SwiftUI's `ScrollView` for a variety of layouts and behaviors. It covers vertical and horizontal scrolling, `LazyVStack`/`LazyHStack` for performance, paging with `scrollTargetBehavior`, and reading scroll position with `scrollPosition`. A practical reference for any scrollable content in SwiftUI.

## ✅ Why you'd use it
- **Lazy stacks** — Shows how `LazyVStack` and `LazyHStack` defer view creation for smooth performance in long lists
- **Paging scroll** — Demonstrates `scrollTargetBehavior(.paging)` and `scrollTargetLayout()` for snap-to-item scrolling
- **Scroll position** — Covers tracking and programmatically controlling scroll offset with `scrollPosition(id:)`

## 🚀 Getting Started

### 1. Clone the Repo
```bash
git clone https://github.com/NDCSwift/ScrollViews.git
cd ScrollViews
```

### 2. Open in Xcode
Double-click `ScrollViews.xcodeproj`.

### 3. Set Your Development Team
In Xcode: **TARGET → Signing & Capabilities → Team** — select your team.

### 4. Update the Bundle Identifier
Change `com.example.MyApp` to a unique reverse-domain ID.

## 🛠️ Notes
- No third-party dependencies required.
- If you see a code signing error, verify Team and Bundle ID are set.

## 📦 Requirements
- Xcode 15+
- iOS 16+
